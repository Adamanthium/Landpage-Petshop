# 🌐 Meu Projeto Frontend

Este é um projeto simples em **HTML, CSS e JavaScript** que demonstra:
- Uso de fontes externas (Google Fonts – Smokum).
- Menu de navegação no header alinhado à direita.
- Footer com ano dinâmico via JavaScript.
- Estrutura organizada para fácil manutenção.

## 🚀 Tecnologias utilizadas
- **HTML5** para estrutura da página.
- **CSS3** para estilização e layout responsivo.
- **JavaScript** para funcionalidades dinâmicas (ano automático no footer).
